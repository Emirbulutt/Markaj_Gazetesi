### Kurulum Adımları

1. Projeyi bilgisayarınıza indirin:

```bash  
  
git  clone [https://github.com/Emirhanbgdyc/Markaj_Gazetesi.git]  
  
```  



2. Proje klasörüne gidin:

```bash  
  
cd [markaj_gazetesi]  
  
```  



3. Gerekli paketleri yükleyin:

```bash  
  
flutter  pub  get  
  
```  



4. Uygulamayı başlatın:

```bash  
  
 flutter  run  
```
